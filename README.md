# Clínica — Frontend

Interface frontend para site institucional de clínica médica, desenvolvida com foco em
experiência do usuário, organização visual, responsividade e clareza na apresentação
dos serviços de saúde.

## Status do Projeto
✅ Frontend finalizado

> Este repositório contempla **exclusivamente o frontend** da aplicação.

---

## Visão Geral
O projeto apresenta um layout moderno e responsivo para uma clínica médica,
incluindo páginas institucionais, listagem de serviços, apresentação de profissionais,
depoimentos de pacientes e seção de blog.

O objetivo é demonstrar competências em desenvolvimento frontend utilizando
framework CSS, estrutura semântica e boas práticas de layout.

---

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Bootstrap Icons

---

## Funcionalidades Implementadas
- Página inicial institucional
- Seção de serviços médicos
- Listagem de especialidades
- Apresentação da equipe médica
- Depoimentos de pacientes
- Seção de blog (layout)
- Layout totalmente responsivo
- Navegação fluida e organizada
